\# HouseHunt MERN Project



HouseHunt is a full-stack MERN (MongoDB, Express, React, Node.js) house rental management system.



\## Features

\- Role-based authentication (Admin, Owner, Renter)

\- Property listing system

\- Booking system

\- Admin dashboard



\## Tech Stack

\- MongoDB

\- Express.js

\- React.js

\- Node.js
